# Unity跑酷模型资源

本仓库提供了一套用于Unity跑酷教程的模型资源。这些资源是制作跑酷游戏时不可或缺的素材，可以帮助开发者快速搭建跑酷场景和角色模型。

## 资源内容

- 跑酷场景模型
- 跑酷角色模型
- 相关道具和障碍物模型

## 使用方法

1. 克隆或下载本仓库到本地。
2. 将资源文件导入到你的Unity项目中。
